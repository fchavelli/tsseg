tsseg.algorithms.bottomup package
=================================

Agglomerative: starts from finest segmentation, iteratively merges the cheapest
pair.

| **Type:** change point detection
| **Supervision:** semi-supervised or unsupervised
| **Scope:** univariate and multivariate
| **Complexity:** O(n log n)

**Implementation:** Vendored from ruptures v1.1.8. BSD 2-Clause.

**Reference:** Keogh, Chu, Hart & Pazzani (2001), ICDM.
