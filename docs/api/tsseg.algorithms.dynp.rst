tsseg.algorithms.dynp package
=============================

Exact optimal partitioning for a fixed number of change points.

| **Type:** change point detection
| **Supervision:** semi-supervised only
| **Scope:** univariate and multivariate
| **Complexity:** O(C Q n²)

**Implementation:** Vendored from ruptures v1.1.8. BSD 2-Clause.

**Reference:** Auger & Lawrence (1989), Bull. Math. Biology.
